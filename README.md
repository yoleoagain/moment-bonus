# Moment bonus

## Instalation

```
npm i
npm run strapi-build
```

## Development

```
npm run backend
npm run dev
```
## Build types from gql schema

```
npm run types
```

## CMS and DB
1. Sign up or Login in http://localhost:1337/admin
2. Set permissons for your user http://localhost:1337/admin/settings/users-permissions/roles
